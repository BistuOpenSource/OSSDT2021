# bookmanager

#### 介绍

图书管理系统（bookmanager），此项目使用了Vue与Springboot，具备分页展示了图书信息、图书检索、图书信息修改与删除功能，使用MIT许可证。


#### 软件架构

逻辑架构

 - vue + springboot

 - vue负责页面渲染

 - springboot负责业务逻辑与数据处理

物理架构

 - 服务器：本地pc
 - 数据库： MySQL-5.8

#### 安装教程

安装好项目的提到的环境，在idea中下载vue插件，先运行后端项目，再运行前端项目。
